# Wanderlust
### Travel. Explore. Experience.

Wanderlust is a full-stack travel listing web application built using Node.js, Express, MongoDB, and EJS.  
The platform allows users to explore, create, and manage travel destinations with secure authentication and a clean, responsive interface.

---

## 🚀 Features
- User authentication & authorization
- Create, edit, and delete travel listings
- Reviews and ratings system
- Image uploads for listings
- Interactive map integration
- Search and category-based filtering
- Responsive UI using Bootstrap

---

## 🛠 Tech Stack
**Backend**
- Node.js
- Express.js
- MongoDB
- Mongoose

**Frontend**
- EJS (Embedded JavaScript Templates)
- Bootstrap
- Vanilla JavaScript

**Other Tools**
- Map integration
- Cloud-based image storage
- RESTful API architecture
